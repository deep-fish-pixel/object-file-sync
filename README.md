# object-file-sync
simple object filesync. This support i18n key's all operations sync.

### Install
Install with npm:

`$ npm install --save object-file-sync`

### 同步文件夹 文件 文件内容key-value操作

```
objectFileSync({
    root: path.join(process.cwd(), 'test/src/locales'),
    // dirs: ['zh-CN', 'en-US'],
});

```
